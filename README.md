# AI-Based Online Exam Proctoring System

## Team Name

[VIGILANCE AI]

## Project Description

AI-Based Online Exam Proctoring System is a real-time online examination monitoring platform developed using Flask, OpenCV, MediaPipe, and Socket.IO.

The system monitors candidates during online exams using webcam-based AI facial landmark analysis. It detects suspicious activities such as:

* Face not visible
* Looking left/right/down
* Suspicious mouth movement

The project also includes an admin dashboard for live student monitoring and warning tracking.
